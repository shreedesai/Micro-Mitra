# Micro Mitra

### AI-Enabled Self-Directed Learning and Competency Reflection Platform for Medical Education

## Overview

**Micro Mitra** is a web-based prototype platform designed to support **Competency-Based Medical Education (CBME)** introduced by the National Medical Commission.

The platform helps medical students and faculty monitor learning progress across three domains:

* **Knowledge**
* **Clinical Skills**
* **Attitude / Communication**

Micro Mitra uses **AI-assisted reflection analysis, adaptive quizzes, and personalized learning roadmaps** to enhance **Self-Directed Learning (SDL)** among medical students.

This repository contains a **functional HTML prototype** demonstrating the core features of the Micro Mitra platform.

---

# Key Features

## 1. Faculty Dashboard

Provides class-level insights including:

* Total student participation
* Average SDL readiness
* Domain progress trends
* Miller’s Pyramid competency distribution
* Subject-wise performance
* Activity monitoring
* Top performing students

---

## 2. Student Self-Directed Learning Portal

Students can access AI-supported learning tools including:

### Reflection Analysis

Students write a reflection about their learning experience.

The system analyzes:

* Learning domain (Knowledge / Skill / Attitude)
* Competency level based on Miller's Pyramid
* Identified learning gaps
* Strengths
* Emotional tone
* SDL readiness score

---

### Personalized Study Roadmap

Based on reflection analysis, the system generates:

* A **4-week personalized study plan**
* Weekly learning objectives
* Suggested activities
* Milestone tracking

---

### Adaptive Quiz Generator

Students can generate quizzes aligned to:

* Subject
* Competency level
* Domain of learning

The quiz module includes:

* Multiple choice questions
* Immediate feedback
* Explanations for answers

---

### AI Tutor

A conversational AI tutor that provides:

* Concept explanations
* Clinical case discussions
* Study guidance
* Exam preparation tips

---

## 3. Admin Panel

The administrative dashboard allows faculty to:

* View student learning analytics
* Identify **at-risk students**
* Monitor competency progress
* Analyze domain-wise performance
* Generate class-level insights

# System Concept

The Micro Mitra system is designed to support **AI-assisted learning analytics**.

Core workflow:

Student Reflection
→ AI Analysis
→ Competency Mapping
→ Gap Identification
→ Personalized Learning Plan
→ Faculty Analytics

# Alignment with CBME

Micro Mitra supports implementation of **Competency-Based Medical Education** by:

* Tracking competency development
* Supporting reflective learning
* Identifying learning gaps
* Promoting self-directed learning

The system also aligns with **Miller’s Pyramid of Clinical Competence**:

1. Knows
2. Knows How
3. Shows How
4. Does
